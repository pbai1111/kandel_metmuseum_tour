# The Neuroscience of Seeing — Met Museum Audio Tour

A self-guided audio tour of The Met Fifth Avenue, pairing artworks with neuroscience quotes from Eric Kandel and cross-book connections from a 215-book personal library.

## Versions

### V1
- 10 stops, European-focused
- 3 voices: Museum (artwork context), Kandel (neuroscience quotes), Library (book connections)

### V2
- 10 stops, cross-cultural (6 continents)
- 4 voices: Museum, Kandel, Library, and Surprise (unexpected connections across disciplines)
- 101 audio clips, ~23 MB

## Built With

- **PKES** (Personal Knowledge Experience System)
- **ElevenLabs TTS** for voice generation
- **Met Open Access API** for artwork data
